# Voronoi Diagram

[Voronoi diagram](https://en.wikipedia.org/wiki/Voronoi_diagram) in Python using pygame.

## Installing pygame

`pip install pygame` (might be pip3 depending on your pip install).

## Launch

`python3 voronoi.py`
